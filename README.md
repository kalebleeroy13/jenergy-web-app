# JEnergy Web Application

A modern, responsive business website built for **JEnergy**, a renewable‑energy consulting and installation company. This project was delivered as a paid freelance engagement and includes a full multi‑page layout, custom branding, and progressive web app (PWA) features.

---

## 🚀 Features

- Fully responsive, mobile‑friendly design
- Multi‑page layout (Home, Services, About, Contact, Portfolio)
- Custom branding and color palette
- Progressive Web App (PWA) support
- Reusable components (header, footer, navigation)
- Clean, maintainable HTML/CSS/JavaScript structure
- Deployed on GitHub Pages
- SEO‑friendly metadata and structure

---

## 🧩 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Service Worker + Web Manifest (PWA)**
- **GitHub Pages** for hosting and deployment

---

## 📘 Case Study

### **Client**
JEnergy — a renewable‑energy consulting and installation business specializing in solar solutions and energy‑efficiency services.

### **Problem**
The client needed a modern, professional website to:

- showcase services
- build credibility
- attract new customers
- provide a clean, mobile‑friendly experience
- serve as their primary online presence

They previously had no website or digital footprint.

### **Solution**
I designed and built a custom multi‑page website with:

- a clean, modern layout
- clear service descriptions
- strong visual identity and branding
- fast loading times
- mobile‑first responsiveness
- PWA support for offline access
- reusable components for easy future updates

### **Outcome**
The client received a polished, production‑ready website and paid **$1500** for the completed project.
The site is now live and used as their primary online presence.
